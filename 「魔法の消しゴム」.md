**「AIが失敗した時に、なかったことにする」**手順

### 手順：変更を破棄（Discard）する

1. GitHub Desktopの左側にある「Changes」リストから、破壊した**ファイル名**を見つけてください。
    
2. そのファイル名の上で、**マウスを「右クリック」**します。
    
3. 出てきたメニューの中から、**「Discard changes...」**（変更を破棄する）を選びます。
    
4. 「本当に破棄しますか？」という確認画面（赤色のボタン）が出るので、迷わず **「Discard changes」** をクリックしてください。
